# Nossa Escola Sem Desperdício de Água

Material de apoio didático criado para uma unidade de ensino que articula
Computação e Educação Ambiental, com foco no uso crítico de tecnologias
digitais e na conscientização sobre o desperdício de água na escola.

## Sobre o projeto

Esta landing page foi pensada para:

- Ser usada em aulas de Computação / Cultura Digital no Ensino Fundamental;
- Dialogar com a habilidade **EF15CO08** da BNCC de Computação;
- Servir como exemplo de campanha digital criada com HTML, CSS e Bootstrap;
- Ser acessível em **computadores e celulares** (layout responsivo).

## Tecnologias

- HTML5
- CSS3
- [Bootstrap 5](https://getbootstrap.com/)
- [Bootstrap Icons](https://icons.getbootstrap.com/)
